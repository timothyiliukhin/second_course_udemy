The Basic Languege of the Web:

All modern websites and web applications are built using three fundamental technologies: HTML, CSS and JavaScript. These are the langueges of the web.

In this post, lets focus on HTML. We will learn what HTML is all about, and why you too should learn it.

What is HTML?

HTML stands for HyperText Makup Language. It's a markup language that web developers use to structure and discribe the content of a webpage (not a programming language).

Html consists of elements that describe different types of content :paraghaps, links, headings, images, video, ets. Web browsers understand HTML and render HTML code as websites.

In Html, each element is made up 3 parts:

  1. The opening tag
  2. The closing tag
  3. The actual element

Why should you learn HTML?

There are countless reasons for learning the fundamental languege of the web. Here are 5 of them:

  • To be able to use the fundamental web dev Languege
  • To hand-craft beautiful web sites instead of relying on tools like Worpress or Wix
  • To build web applications
  • To impress friends
  • To have fun
